# 🎲 Full Stack Ludo Game

A real-time multiplayer Ludo game developed using modern full-stack technologies. Players can create or join game rooms, roll dice, move tokens, and play with friends in a responsive and interactive interface.

## 🚀 Features
- 🎮 Real-time multiplayer gameplay
- 🔐 User Authentication & Authorization
- 🏠 Create and Join Game Rooms
- 🎲 Dice Roll Logic
- ♟️ Token Movement with Ludo Rules
- 🔄 Live Game State Synchronization
- 📱 Responsive Design
- 📊 Score & Winner Tracking

## 🛠️ Tech Stack
- Frontend: React.js, HTML, CSS, JavaScript
- Backend: Java, Spring Boot
- Real-time: WebSocket
- Version Control: Git & GitHub
